# Security Policy

## Supported Versions
Only the latest release is supported with security updates.
Older releases are not supported for security fixes.

## Reporting a Vulnerability
If you discover a potential security vulnerability, report it by email:
`security@attentionfirst.dev`

- Do NOT open a public GitHub issue for security vulnerabilities.
- Include a clear description of the vulnerability.
- Include exact steps to reproduce.
- Include the affected version.
- Include the potential impact.

### Response Timeline
- We acknowledge reports within 48 hours.
- We provide an initial assessment within 7 days.

### Disclosure
We follow coordinated disclosure with a 90 day window.
